Teste sua aplicação com o Disguise

Facilite o uso de stubs e mocks fornecidos pelo Mountebank Server.

Interface amigável

Feito para tornar o uso do Mountebank Server mais simples para desenvolvedores.

Configuração rápida

Faça download e upload de arquivos para configuração de testes.

Centralização de informações

Encontre as principais funcionalidades para os testes em um único lugar.

Funcionalidades

  • Crie impostores

    Configure os stubs e tudo que é necessário para utilizar o Mountebank.

  • Visualize logs

    Acesse todas as mensagens retornadas pelo servidor.

  • Importação e exportação de arquivos

    Faça upload e download de arquivos de configuração JSON.

Como usar o Disguise

Para utilizar o Desguise, instale o Mountebank Server seguindo os passos abaixo:

1

Node.js

Instale o Node.js Version 12. Para acessar a página de download clique aqui.

2

Mountebank

Instale o Mountebank. Para acessar a documentação clique aqui.

3

Impostores

Gerencie seus impostores.Com o Mountebank Server em execução, crie e administre impostores e stubs.

4

Logs

Visualize os logs.Acesse com facilidade os logs gerados pelo Mountebank Server.
